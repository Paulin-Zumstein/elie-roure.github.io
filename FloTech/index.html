Flot-V1

Présentation

Le Flot-V1 est un flotteur électronique destiné à tous les pêcheurs. Grâce à sa technologie ainsi que ses multiples capteurs, il permet aux nouveaux adeptes de cette passion de posséder tous les moyens pour progresser rapidement. Pour les adeptes, il va leur permettre de gagner un temps considérable en collectant de lui-même toutes les informations nécessaires à une pêche de qualité. Le Flot-V1 n'est pas qu'un flotteur électronique, c'est aussi une application mobile pour pouvoir lire les données du flotteur.

Caracteristiques

Le Flot-V1 c'est : un flotteur en bois, une réserve à appât, une application mobile, un thermomètre, un capteur ph, une turbine avec un capteur à effet ha, un détecteur de distance et un module radio.
Toute cette technologie vous permet d'avoir sur votre téléphone grâce à l'application mobile, le taux de salinité ainsi que le pH de l'eau. La turbine couplée avec un capteur à effet ha vous permet d'avoir la force du courant pour bien déterminer les poissons que vous pouvez trouver. Un thermomètre accompagne le tout pour plus de précision sur l'environnement péché. Une encyclopédie est intégrée à l'application mobile pour pouvoir utiliser le plein potentiel des informations récoltés.
